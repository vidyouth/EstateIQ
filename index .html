About EstateIQ

A smart real estate portal powered by machine learning.

🎯 Aim

Design and develop a smart real estate portal that uses a Decision Tree algorithm to predict property prices and recommend suitable properties to users, while enabling direct buyer–seller communication.

🔬 Technology

The core ML model analyses attributes like location, area, rooms, and property age using a decision tree to predict fair market value. This reduces manual effort and enhances transparency.

👥 User Roles

  • Buyer — Search, filter, predict, and chat with sellers.
  • Seller — List properties, respond to inquiries, manage listings.
  • Admin — Full platform control, moderation, analytics.

🏗️ Development Model

Built using the Waterfall Model — a structured, phase-wise approach covering requirement analysis, system design, implementation, testing, deployment, and maintenance.
